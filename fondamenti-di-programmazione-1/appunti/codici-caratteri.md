# Caratteri e codici

I computer memorizzano i caratteri come se fossero numeri interi.

In Python sono disponibili due funzioni per ottenere il codice di un carattere e il carattere associato a un codice.

```python
>>> ord("a")
97
>>> ord("b")
98
>>> ord("c")
99
>>> ord("d")
100
```

```python
>>> chr(50)
'2'
>>> chr(49)
'1'
>>> chr(97)
'a'
>>> chr(75)
'K'
```
